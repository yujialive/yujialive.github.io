## PyCharm 快捷键

- Ctrl + Space　# 语法智能提示
- Ctrl + K  # Commit Change dialog
- Double Shift # Find any action, symbol and setting
- Ctrl + Click Window # toggle vertical and side by side placement
- Ctrl + Alt + Shift + T # refact menu-
